Setting up Backend on Mac:
1. Install mongodb using homebrew and install the compass GUI.
2. Set up the .env file
3. Start the mongo services
- brew services start mongodb-community@7.0
4. Run the backend using npm run dev
5. Stop mongo services when stopping backend too
- brew services stop mongodb-community@7.0

To export and import data
mongodump -d webApp
